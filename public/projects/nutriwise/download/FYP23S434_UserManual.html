Running the Application

Opening the project in the IDE of your choice

  1. We recommend using VSCode to when opening the project
  2. Open up 2 terminals, one will be used from the Frontend (FE) and the other used for the Backend (BE)

Running the Frontend (FE)

  1. In the terminal for FE, make sure that you are in the root folder and run the following commands:

npm i

npx expo start

  1. Wait for Expo Go to load and you should see a QRCode as well as options for you to pick from
  2. If you are using an iPhone with ExpoGo installed scan the QRCode using the camera app iPhone
  3. If you are using the XCode simulator, press i to run the app in the iOS environment
  4. Make sure the devices are in the same Wi-Fi as your servers
  5. The simulator should load on your device and you should be redirected to the Welcome Page

Running the Backend (BE)

  1. In the terminal for BE, navigate to the api folder by running the following command

cd api

  1. Once in the api folder, run the following commands:

npm i

npm start



  1. Wait for NodeJs to load and you should see the following in your terminal

Listening on Port: 8000

Db is connected

Explore the Application

  1. Once your FE and BE are up and running, you will be able to explore the application with all its features included
  2. Ensure that you have a good internet connection to reduce latency or potential timeouts between pages

Navigating the Application

1

Startup Page

Upon loading up the application, you will be greeted with the following page.

To register for a new account, clicking on ‘Create Account’ will take you to our Sign Up page. Proceed to 2 Sign Up Page.

Otherwise to login, click on ‘Let’s Get Started’. Proceed to 3 Login Page.

2

Signup Page

Upon clicking on ‘Create Account’, you will be taken to this page.

Input a valid username, email and password. Click on ‘Create Account’ to proceed.

Otherwise, click on ‘Log In’ to login instead. Proceed to .

2.1

Enter Your Particulars

Upon clicking on ‘Create Account’, you will be taken to this page:

Input your name, country and a valid phone number. Click on the arrow button to proceed.

Otherwise, click on ‘Back’ to go back.

2.2

Get Your Measurements

Upon clicking on the arrow button, you will be taken to this page:

Input your age, gender, weight and height. Click on the arrow button to proceed.

Otherwise, click on ‘Back’ to go back.

2.3

Select Your Diabetes Type

Upon clicking on the arrow button, you will be taken to this page:

Select your diabetes type, then click on the arrow button to proceed.

Otherwise, click on ‘Back’ to go back.

3

Login Page

Upon clicking on ‘Log in’, you will be taken to this page:

Input your account email and password. Click on ‘Log In’ to proceed. Proceed to 4 Dashboard Page.

Otherwise, click on ‘Sign Up’ to sign up. Proceed to 2 Sign Up Page.

4

Dashboard Page

Upon logging in or signing up, you will be taken to this page:

This is the dashboard page, displaying the user’s calorie and glucose levels.

The calories table will display the amount of calories consumed and the amount of calories left from their target. By clicking on it, you will be taken to 8.3 Calorie Summary Page.

The glucose table will display the latest glucose level input and the average glucose level for the day. By clicking on it, you will be taken to 9.2 Calorie Summary Page.

At the bottom of the page is a taskbar with three buttons. The ‘Home’ button takes you to the dashboard page.

The ‘+’ button opens up a menu. Proceed to 5 Menu Page.

The ‘Profile’ button takes you to the profile page. Proceed to 6 Profile Page.

5

Menu Page

Upon clicking on the ‘+’ button, you will be taken to this page:

The user will be presented with 8 options.

The first option ‘Profile’ takes you to the profile page. Proceed to 6 Profile Page.

The second option ‘Medication’ takes you to the medication page. Proceed to 7 Medication Page.

The third option ‘Food Diary’ takes you to the food diary page. Proceed to 8 Food Diary Page.

The fourth option ‘Glucose’ takes you to the glucose page. Proceed to 9 Glucose Page.

The fifth option ‘Analysis’ takes you to the analysis page. Proceed to 10 Analysis Page.

The sixth option ‘Healthcare Info’ takes you to the healthcare info page. Proceed to 11 Healthcare Info Page.

The seventh option ‘Glucose Info’ takes you to the glucose info page. Proceed to 12 Glucose Info Page.

The eighth option ‘Settings’ takes you to the settings page. Proceed to 13 Settings Page.

6

Profile Page

Upon clicking on the ‘Profile’ button, you will be taken to this page:

This is the profile page, displaying the user’s name, age, diabetes type and a table with measurements and glucose levels.

The table displays the user’s basic measurements such as height, weight and calorie goal, and the user’s blood glucose levels such as Hypo, acceptable range and Hyper.

7

Medication Page

Upon clicking on ‘Medication’, you will be taken to this page:

This is the medication page, displaying the user’s medication. The user will be able to click on the pen icon to edit their medication details. Otherwise, the user can click on the green tick button to confirm consumption of their medication.

Lastly, at the bottom is the ‘Add Medication’ button. Proceed to 7.1 Add Medication Page.

7.1

Add Medication Page

Upon clicking on the ‘Add Medication’ button, you will be taken to this page:

This is the add medication page. The user will be able to input their medication name, dosage level, and time.

At the bottom is a confirm button for the user to save their medication details.

8

Food Diary Page

Upon clicking on “Food Diary’, you will be taken to this page:

This is the food diary page, displaying the user’s breakfast, lunch and dinner. At the bottom are two buttons.

The left option ‘New Entry’ takes you to the new entry page. Proceed to 8.1 New Food Entry Page.

The right option ‘Scan A Barcode’ which opens up the camera application for the user to scan a barcode.

8.1

New Food Entry Page

Upon clicking on ‘New Entry’, you will be taken to this page:

At the top of the page is a search bar. The user will be able to search for any food items that have been consumed.

Upon entering a food item, search options will drop down for the user to select.

Select a search result and you will be taken to the next page. Proceed to 8.1.1 Food Entry Summary.

8.1.1

Food Entry Summary

Upon selecting a search result, you will be taken to this page:

This is the summary page, displaying the food’s nutritional details such as calories, carbs, fat, protein and fibre. The user will also be able to enter the serving size and number of servings.

At the bottom of the nutritional table is a green tick button. Click on the button to complete the meal entry. You will be taken back to 8 Food Diary Page.

Below the nutritional table is the glucose prediction table. The user will be recommended an insulin dose based on their intake. Click on the green tick button to add the insulin amount into 7 Medication Page.

8.2

Calories Summary Page

Upon clicking on the calories table in the home page, you will be taken to this page:

It shows the user’s calories intake for the day, week and month.

9

Glucose Page

Upon clicking on the ‘Glucose’ button, you will be taken to this page:

This is the glucose records page, displaying the user’s glucose records.

The table displays the blood glucose levels and the time when it was taken.

The bottom button ‘New Records’ takes you to the new entry page. Proceed to 9.1 New Glucose Record Page.

9.1

New Glucose Record Page

Upon clicking on the New Records button, you will be taken to this page:

In the middle of the page is a table indicating the date and time. The user is able to enter the meal type and glucose levels.

At the bottom of the glucose table is a green tick button. Click on the button to complete the glucose entry. You will be taken back to 1.9 Glucose Page.

Below the glucose table is the recommended insulin table. The user will be recommended an insulin dose based on their glucose level. Click on the green tick button to add the insulin amount into 7 Medication Page.

9.2

Blood Glucose Summary Page

Upon clicking on the glucose table in the home page, you will be taken to this page:

It shows the user’s blood glucose levels for the day, week and month. There is also an analysis at the bottom.

10

Analysis Page

Upon clicking on the Analysis button, you will be taken to this page:

It displays an analysis of the user’s caloric progress and blood glucose progress.

At the bottom of the page is a ‘send email’ button. The user will be able to send his analysis to others.

11

Healthcare Info Page

Upon clicking on the Healthcare Info button, you will be taken to this page:

It displays the details of a healthcare provider.

The user will be able to add or edit details of the healthcare provider.

12

More Info Page

Upon clicking on the More Info Button, you will be taken to this page:

It displays important information for diabetics to read. There is information regarding hypoglycemia, hyperglycemia, food portioning to name a few.

There is also an events tab that displays events to the users to attend. The products tab displays products that are beneficial to our users.

13

Settings Page

Upon clicking on the Settings button, you will be taken to this page:

It allows the user to select 5 options:

The first option is ‘Profile’. Proceed to 13.1 Edit Profile Page.

The second option is ‘Nutritional Details’ Proceed to 13.2 Edit Nutritional Details Page.

The third option is ‘Blood Glucose Details’. Proceed to 13.3 Edit Blood Glucose Details Page.

The fourth option is ‘Emergency Contact Details’. Proceed to 13.4 Update Emergency Contact Details Page.

The fifth option is ‘Logout’. The user will be able to logout from the application.

13.1

Edit Profile Page

Upon clicking on the Profile button, you will be taken to this page:

The user will be able to edit their first name, last name, age, gender, height and weight.

At the bottom is a confirmation button for the user to confirm their changes.

13.2

Edit Nutritional Details Page

Upon clicking on the Nutritional Details button, you will be taken to this page:

The user will be able to edit their calorie goal, fat percentage, carbs percentage and protein percentage.

At the bottom is a confirmation button for the user to confirm their changes.

13.3

Edit Glucose Details Page

Upon clicking on the Glucose Details button, you will be taken to this page:

The user will be able to edit first name, last name and age.

At the bottom is a confirmation button for the user to confirm their changes.

13.4

Edit Emergency Contact Page

Upon clicking on the Emergency Contact button, you will be taken to this page:

The user will be able to edit the first name, last name, email and contact number of their emergency contact.

At the bottom is a confirmation button for the user to confirm their changes.
